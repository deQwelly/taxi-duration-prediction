# Taxi Duration Prediction

## Обзор проекта
В данном проекте реализована модель машинного обучения для предсказания длительности поездок такси на основе исторических данных.

Основная цель — построить воспроизводимый и масштабируемый ML-пайплайн.

Была произведена работа, охватывающая полный цикл работы с данными:
- разведочный анализ
- подготовка данных
- разработка признаков
- обучение моделей
- оценка качества

Проект ориентирован на получение как можно более точного предсказания, а также на создание решения, пригодного для использования в production.

---

## Постановка задачи
Задача заключается в предсказании времени поездки (trip duration) на основе параметров поездки.

Такие модели могут использоваться для:
- расчета времени прибытия
- поддержки систем динамического ценообразования
- оптимизации операционных процессов
- улучшения пользовательского опыта

Задача формализуется как задача регрессии.

---

## Данные
Набор данных содержит информацию о поездках такси:
- время начала поездки
- координаты посадки и высадки
- количество пассажиров
- и другие переменные

Целевой фактор — длительность поездки.

---

## Разведочный анализ данных
В рамках анализа были выполнены:
- исследование распределения целевой переменной (в т.ч. для разных значений других переменных)
- анализ временных зависимостей
- анализ географических факторов
- выявление выбросов и аномалий

Основные наблюдения:
- распределение длительности ассиметрично, наблюдаются тяжелые хвосты
- присутствуют выбросы, требующие фильтрации
- длительность поездки более всего зависит от времени и расстояния

---

## Предобработка данных
Были выполнены следующие шаги:
- заполнение пропусков в данных
- приведение типов данных
- фильтрация нереалистичных значений по:
    - длительности поездки
    - расстоянии от посадки до высадки
    - регрессионным остаткам

Предобработка спроектирована с учетом повторного использования на этапе инференса.

---

## Генерация признаков
Были разработаны следующие группы признаков:

Временные признаки:
- час
- день недели
- месяц
- день в году

Географические признаки:
- Манхэттенское расстояние между точками (с использованием формулы Haversine)
- область посадки и высадки
- топ 100 по области посадки и высадки
- флаг поездки из/в особое место (аэропорты, порты и т.д.)

Дополнительные признаки:
- номер поставщика
- количество пассажиров
- флаг отложенной передачи данных поездки
- флаг свободной/загруженной дороги

Категориальные признаки были закодированы с помощью One Hot Encoding (OHE)

Генерация признаков является ключевым фактором повышения качества модели.

---

## Моделирование
Рассмотрены и сравнены несколько моделей:
- Ridge регрессия
- LASSO регрессия

В процессе обучения:
- использовано разбиение на обучающую и тестовую выборки
- проведена настройка гиперпараметров с использованием кросс-валидации

---

## Оценка качества
Основная метрика:
- RMSLE (Root Mean Squared Logarithmic Error)

Причины выбора:
- устойчивость к выбросам
- учет относительной ошибки

Дополнительно проведены:
- анализ распределения ошибок
- проверка стабильности модели
